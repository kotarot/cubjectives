• Cubes in Rubik's Cube: Multiple Blindfolded means difference (# cubes solved - # cubes missed), so larger is better.
  • The result is based on the simple mathematics: Correlation and regression line (calculated by method of least squares).
  • The source codes are available at GitHub repository. If you get interested in this system and have ever touched Web technologies (especially Python, MySQL, HTML, javascript, ...), feel free to fork it, modify it as you like, and send pull requests to me.
  • Details of the system are described in my Japanese blog article (will be updated soon).
  • Since I have not tried any tests on IE, use a modern web-browser such as Firefox, Google Chrome, and Safari.
